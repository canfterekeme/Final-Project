# Final-Project
Trigonometric Chirp Based Multi-User Spread  Spectrum Communication System

ABSTRACT 
In this research project, a multiuser communication system is proposed assigning the same 
carrier frequency and almost the same frequency range for each user in the same band to improve 
the spectral efficiency and reduce the Multi-Access Interference (MAI) in the limited bandwidth 
conditions. Since an analytical description of the signals transmitted by different users is required 
in the case of sharing the same frequency carrier and the same bandwidth over a fixed duration of 
bits, the Trigonometric Chirp Signal is used for transmitting the signals. Furthermore, this system 
offers orthogonal properties regardless of the Time-Bandwidth (TB) product values are variable 
or high as compared to Linear Frequency Modulation LFM systems. 
In addition, one of the major issues faced in multi-user Frequency Modulation ( FM) systems 
sharing the same carrier frequency is the power superimposed on the carrier frequency, therefore, 
a specific value of modulation index should be chosen so as to have zero power on the carrier 
frequency (i.e., suppressed carrier). Moreover, due to allocation of baseband frequency to each 
user in such a way that ensures that no spectral overlapping occurs up to the bandwidth where the 
power of the high-order harmonics becomes negligible; this method eliminates the necessity for 
windowing in the frequency domain, thereby reducing the complexity of receivers. 
Performance of the system will be evaluated using Inner Product Analysis, Cross-Correlation 
Analysis in both Time Domain and Frequency Domain and Bit Error Rate (BER) Analysis to show 
the simulated results. Therefore, this new architecture represents a novel concept in existing 
literature by combining the features of perfect orthogonality at variable time-bandwidth (TB) 
products, identification of users based on frequency of their harmonic, and improved spectral 
efficiency of the system due to the natural sideband effects. Hence, it has the potential to be utilized 
as a strategic alternative for communication systems which require high capacities and low error 
rates in bandwidth limited environments.
